Python Turtle Game Project


We first import the turtle and the random module. After that is complete, we establish the screen’s background

color and our pensize and pen color, along with the turtle shape and its initial size. We then add what constitutes

an apple in our game, making it a red circle. We also add in a color list. Then move onto making the scoring system.

Starting it at 0 and putting the score’s location on the screen in the upper right corner. Next, we set up a timer

which runs for a total of 30 seconds and appears in the upper left corner. Afterwards we create the process that

enables the apple to randomly appear on the map after the collision with the turtle, as well as how much the turtle

grows after eating it (0.2). Past that, we enable the turtle’s trail to change color and the game to end once the timer

runs out. Finally, we established the arrow key movements that allow the turtle to be moved in the first place.



The Cup Game project


This was a game similar to the Shell Game. It would first ask you for you name, then give you different options like

what color would you like the ball, and what difficulty/speed you would want the cups to move. After you answered the

ball would go into one of the cups and move. You would try to follow the cup and guess where the ball was. If you guess

correctly, you get to continue and have the option to change difficulty. However, if you get it incorrectly, the game

will display the leaderboard screen and you name will be on if you got onto the leaderboards.


The Scratch project

Scratch game

We created a scratch game where you control a basketball by using arrow keys to move it up and down. Different objects

will fly at you and you will try to avoid them. Each object that you successfully dodge will increase you score by one.

When you do get hit by an object, everything will freeze and a game over screen will be displayed.
